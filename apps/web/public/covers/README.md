# Ink book covers

Drop the six pen-and-ink cover images here, named **exactly** as below
(all lowercase, `.png`). They are served from the site root at `/covers/<id>.png`.

| Save as                              | Book                          |
| ------------------------------------ | ----------------------------- |
| `feast-of-the-broadcast-saints.png`  | Feast of the Broadcast Saints |
| `what-survives-is-proof.png`         | What Survives Is Proof        |
| `declensions-of-dark-water.png`      | Declensions of Dark Water     |
| `terms-of-unbeing.png`               | Terms of Unbeing              |
| `summer-of-the-glass-bees.png`       | Summer of the Glass Bees      |
| `the-quiet-metric.png`               | The Quiet Metric              |

Until a file exists, the site falls back to the previous remote cover for that
book, so nothing breaks. PNG preferred; the art has a white background.
